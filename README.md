# Hatfield_Financial
financial advisor that recommends stocks. This uses react


## Setup

This project uses a Python virtual environment (`.venv`). Activate it before running the backend.

### Backend (Flask API on port 5000)

```bash
cd Backend
..\.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

### Frontend (React on port 3000)

```bash
cd Frontend
npm install
npm start
```

Run these in separate terminal windows so both can run simultaneously.

### Verify

- Backend: http://localhost:5000
- Frontend: http://localhost:3000
